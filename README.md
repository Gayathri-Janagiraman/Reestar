# Reestar

Reestar is a basic restaurant webpage showcasing delicious food items including pancakes, noodles, pizza, and desserts. This project includes an `index.html` for the structure, `style.css` for the styling, and attractive food images sourced from Google.

## Features

- **Homepage**: A welcoming layout displaying various food items.
- **Images**: High-quality food images for visual appeal.
- **Responsive Design**: Optimized for different screen sizes using CSS.

## How to Use

1. Clone the repository.
2. Open `index.html` in your web browser.

## File Structure

- `index.html`: Main structure of the webpage.
- `style.css`: CSS styling for the webpage.
- `images`: some food images from google.

## Food Items

- **Pancakes**: Fluffy pancakes with maple syrup and fresh fruit.
- **Noodles**: Savory noodles with fresh vegetables and sauces.
- **Pizza**: Handcrafted pizzas with crispy crust and fresh toppings.
- **Desserts**: Decadent desserts for a sweet finish.
